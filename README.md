# zh-portal-journal

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容彼此独立，不针对任何特定域名或具体网站。

## 项目简介

zh-portal-journal 是一个轻量级的页面归档项目，主要目的是将分散的 HTML 页面集中存放，便于管理和访问。每个页面均以完整的 HTML 文件形式存在，无需依赖外部数据库或后端服务。

## 目录说明

```
zh-portal-journal/
├── pages/           # 存放所有 HTML 页面文件
├── assets/          # 公共资源文件（CSS、JavaScript、图片等）
├── index.html       # 仓库首页 / 页面索引
└── README.md        # 本文件
```

- **pages/**：每个子目录或文件对应一个独立的页面归档。
- **assets/**：存放页面引用的样式、脚本及媒体资源。
- **index.html**：提供页面列表或导航入口，方便浏览所有归档内容。

## 页面归档说明

- 所有页面均为独立的 HTML 文件，可直接在浏览器中打开。
- 页面内容可能包含图片、样式、脚本等资源，均保存在 `assets/` 目录下。
- 归档内容保持原始状态，不进行修改或更新，仅作资料保存用途。
